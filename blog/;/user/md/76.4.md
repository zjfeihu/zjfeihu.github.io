尾递归

##定义
函数中所有递归形式的调用都出现在函数的末尾，并且递归调用不属于表达式的一部分。


##示例
code.js
alert(f1(15) == f2(15, 1))
function f1(n){
    return n < 2 ? 1 : f1(n - 1) * n;
}

function f2(n, result){ //尾递归
    return n < 2 ? result : f2(n - 1, result * n);
}
.


##特征
2. 递归调用为单一的函数调用，不参与运算
1. 中间运算结果作为参数传入下次递归调用
2. 最终运算结果作为递归结束的返回值

info 比起传统的递归，多了一个中间结果传递的过程，使得环境可以跳过保存中间运算结果的过程。
