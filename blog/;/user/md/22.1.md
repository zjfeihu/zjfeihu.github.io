API之classList##前言classList是html5中新增加的API，扩展节点的class操作。##API通过Element.classList来获取classList对象###属性+ length 返回className的个数###方法+ add(className) 添加一个className+ contains(className) 判断是否包含className+ item(index) 返回指定索引的className，从0开始，找不到返回null+ remove(className) 移除指定的className+ toggle(className) 切换指定的className##演示代码code.html<!doctype html><style>.red{color:red;}</style><div id="testbox">testbox</div><button id="add">add</button> <button id="remove">remove</button> <button id="toggle">toggle</button> <button id="contains">contains</button> <button id="item">item</button> <script>var classList = $('testbox').classList;$('add').onclick = function(){    classList.add('red');};$('remove').onclick = function(){    classList.remove('red');};$('toggle').onclick = function(){   classList.toggle('red');};$('contains').onclick = function(){   alert(classList.contains('red') ? '包含red' : '不包含red');};$('item').onclick= function(){   alert('item(0) == ' + (classList.item(0) === null ? 'null' : classList.item(0)));};function $(id){    return document.getElementById(id);}</script>.##兼容性除了IE6-9之外的主流浏览器都能用。
