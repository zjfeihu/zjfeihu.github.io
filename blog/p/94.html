总结得出的换算公式

offsetHeight = height + padding + border
clientHeight = height + padding - bottomScroll（底部滚动条所占空间）
scrollHeight = Math.max( height + padding, inner.offsetHeight )

在使用outer.scrollHeight的场景中，规避使用inner.margin和outer.padding
