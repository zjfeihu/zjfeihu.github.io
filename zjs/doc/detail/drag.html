drag( )

原型方法 define in : widget

语法

  • drag( option ) 创建拖放对象

参数

  • option [Object]

    必选 拖拽配置对象

  • option配置参数

    key 类型 说明
    before function 拖动前回调
    running function 拖动中回调
    after function 拖动完成后
    limitNode Node 限制范围的节点，参数为window则限定为浏览器可见屏幕内
    lockx 1|0 锁定横向拖动
    locky 1|0 锁定纵向拖动
    hand Node|NodeList 拖动的句柄容器

  • 接口说明

    值 描述
    addHand 添加拖放句柄
    rmvHand 移除拖放句柄，参数为节点

返回值

拖拽对象 [Drag]
