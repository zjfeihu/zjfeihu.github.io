css( )

原型方法 define in : dom.style

语法

  • css( name ) 获取样式

  • css( name, value ) 设置样式

  • css( csstext ) 批量设置样式

  • css( data ) 批量设置样式

参数

  • name [String]

    必选 获取样式名称为name的属性值

  • value [String]

    必选 设置样式，value为值，可以使用中划线也可以采用驼峰

  • csstext [String]

    必选 设置css，标准的csstext字符串，比如margin-left:12px;color:red;

  • data [Hash]

    必选 批量设置css，key作为样式名，value作为样式值

返回值

样式值，Z对象 [String,Z]
