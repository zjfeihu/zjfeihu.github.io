cookie( )

define in : bom

语法

  • cookie( key ) 获取cookie的值

  • cookie( key, value, option ) 设置cookie的值，当value为null时，则删除对应的cookie

  • cookie( data, option ) 批量设置cookie的值

参数

  • key [String]

    必选 cookie的名称

  • value [String]

    必选 cookie的新值

  • data [Hash]

    必选 一组cookie的新值

  • option [Hash]

    可选 设置cookie的有效期等其他属性

  • option说明

    字段 说明
    expires cookie有效期，默认值是直到浏览器关闭时失效，单位是分钟
    path cookie的路径范围，默认是根目录
    domain cookie的domain权限，默认是当前domain

返回值

cookie的值 [String,Null]
