ajax( )

define in : request

语法

  • ajax( option ) 创建一个功能丰富的ajax，比get和post更多配置项

参数

  • option [Hash]

    必选 配置参数

  • option配置说明

    key 描述
    url 请求地址
    type 请求类型，get或post
    data 发送的数据，缺省null
    callback 回调函数，函数的参数由responseType决定，默认是xhr对象
    responseType 响应的类型，决定callback的参数类型
    headers 设置请求头
