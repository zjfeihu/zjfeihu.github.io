append( )

原型方法 define in : dom.operate

语法

  • append( element, index ) 添加子节点

  • append( htmlstr, index ) 添加子节点

参数

  • element [Z,Element]

    必选 待添加的节点

  • htmlstr [String]

    必选 待添加的html字符串

  • index [Integer]

    必选 子元素插入的位置，默认为-1，即添加到最后，取值可以是正数或者负数，正数超出则添加到尾部，负数超出则添加到头部

返回值

Z对象 [Z]
